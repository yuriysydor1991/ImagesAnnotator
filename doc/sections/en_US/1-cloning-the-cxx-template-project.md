# Cloning the ImagesAnnotator project

In order to build the ImagesAnnotator application clone this project into your local directory by executing next command in the terminal (GNU/Linux based):

```
git clone https://github.com/yuriysydor1991/ImagesAnnotator.git
```

or

```
git clone https://gitlab.com/yuriysydor1991/imagesannotator.git
```

After this command executes you should find a directory named `ImagesAnnotator` inside your current one. So enter it by calling `cd ImagesAnnotator`.

The project was fast-started with the help of the C++ templated project located at [https://github.com/yuriysydor1991/cpp-app-template](https://github.com/yuriysydor1991/cpp-app-template).
