# Клонування проекту ImagesAnnotator

Для того щоб побудувати програму ImagesAnnotator необхідно клонувати даний проект у локальну директорію за допомогою наступної команди у терміналі (для GNU/Linux ОС і подібних):

```
git clone https://github.com/yuriysydor1991/ImagesAnnotator.git
```

або

```
git clone https://gitlab.com/yuriysydor1991/imagesannotator.git
```

Після виконання даної команди у поточній директорії повинна з'явитись директорія з назвою `ImagesAnnotator`. Отож необхідно перейти у неї за допомогою команди `cd ImagesAnnotator`.

Реалізація проекту значно пришвидшена завдяки C++ проекту-шаблону розміщеного за адресою [https://github.com/yuriysydor1991/cpp-app-template](https://github.com/yuriysydor1991/cpp-app-template).
